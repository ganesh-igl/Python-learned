# Python-learned
# 🐍 Python Lab Programs
 
> A collection of beginner-to-intermediate Python programs covering core programming concepts — from basic I/O to file handling, matrices, and number theory.
 
---
 
## 📂 Programs
 
| # | File | Description |
|---|------|-------------|
| 01 | `HELLO_WORLD.py` | The classic first program — prints "Hello World" to the console. A clean, commented starting point for any Python journey. |
| 02 | `Arithmetic_Operations.py` | Takes two integers as input and displays their sum, difference, product, quotient, and average using formatted f-strings. |
| 03 | `Largest_of_Three_Numbers.py` | Reads three integers and determines the largest using Python's built-in `max()` function. Concise and beginner-friendly. |
| 04 | `Celsius_to_Fahrenheit.py` | Converts a temperature from Celsius to Fahrenheit using the standard formula, displaying the result rounded to two decimal places. |
| 05 | `Fahrenheit_to_Celsius.py` | Converts a temperature from Fahrenheit to Celsius using the inverse formula, displaying the result rounded to two decimal places. |
| 06 | `Leap_Year_Check.py` | Checks whether a given year is a leap year using divisibility rules for 400, 100, and 4 — handles all edge cases correctly. |
| 07 | `Factorial_Using_Loop.py` | Computes the factorial of a given non-negative integer using an iterative loop approach, avoiding recursion overhead. |
| 08 | `Fibonacci_Series_Using_Loop.py` | Generates and prints the Fibonacci series up to N terms using a loop with efficient tuple-swap variable updates. |
| 09 | `Prime_Number_Check.py` | Determines if a given number is prime by trial division up to its square root — handles edge cases like numbers less than 2. |
| 10 | `Reverse_a_Given_Integer.py` | Reverses the digits of a given integer, correctly preserving the sign of negative numbers using string slicing. |
| 11 | `Palindrome_Check_(String).py` | Checks if a string is a palindrome by comparing it with its reverse using Python's slice notation `[::-1]`. |
| 12 | `Palindrome_Check_Using_String_Library.py` | An enhanced palindrome checker that strips spaces and normalizes case before comparison — handles phrases like "A man a plan". |
| 13 | `Count_Vowels_and_Words_in_a_String.py` | Counts the total number of vowels and words in a user-input string using `sum()` with a generator and `str.split()`. |
| 14 | `Grade_Calculator.py` | Calculates total marks and percentage across 5 subjects, then assigns a letter grade (A–F) based on percentage thresholds. |
| 15 | `Array_Operations_(max_min_sum_avg).py` | Reads N integers into a list and displays the maximum, minimum, sum, and average using Python's built-in functions. |
| 16 | `Roots_of_a_Quadratic_Equation.py` | Solves a quadratic equation using the discriminant — handles two real roots, one repeated root, and complex roots via `cmath`. |
| 17 | `Matrix_Addition_Subtraction_Multiplication.py` | Performs addition, subtraction, and multiplication on two user-defined matrices; multiplication is skipped for non-square inputs. |
| 18 | `Menu-Driven_Calculator_(Switch-Case).py` | A menu-driven calculator using Python's `match-case` statement supporting addition, subtraction, multiplication, and safe division. |
| 19 | `File_Create_and_Write_Operations.py` | Creates a text file, writes student data (name, roll, subject, department), and reads it back — demonstrates basic file I/O. |
| 20 | `Copy_Content_from_One_File_to_Another.py` | Copies content from `student_data.txt` to a new file, then prints the copied content — includes error handling for missing source. |
| 21 | `Size_of_Data_Types.py` | Uses `sys.getsizeof()` to display memory sizes of Python's core data types — int, float, str, and bool — in bytes. |
| 22 | `Sort_Array_in_Ascending_Order.py` | Sorts an array of N integers in ascending order using the bubble sort algorithm, mirroring the traditional C-style approach. |
| 23 | `String_Operations_Using_Library_Functions.py` | Demonstrates common string operations — length, concatenation, comparison, uppercase, and reverse — on two user-input strings. |
| 24 | `Student_Details_Using_Structure_(Class).py` | Models a student record using a Python class with name, roll, and marks attributes, demonstrating OOP fundamentals with a `display()` method. |
| 25 | `Swap_Two_Variables_Using_a_Function.py` | Swaps two integers via a function that returns a tuple, showcasing Python's clean multiple-return-value syntax before and after swap. |
| 26 | `Transpose_of_a_Matrix.py` | Reads an r×c matrix, prints the original, then computes and displays its transpose using list comprehension. |
| 27 | `Triangular_Patterns_Using_Nested_Loops.py` | Prints three star/number patterns — right-angle triangle, inverted triangle, and number triangle — using nested loops. |
